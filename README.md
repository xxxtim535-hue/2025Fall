# 2025fall

这是yh的数算总结

### 1、介绍：

2025fall为大二上数算的课程文件，作业，练习代码，并没有分类，因为自己做的题目不多cheatsheet是自己在考试前的总结和学习，内容也不多

### 2、学习总结：

不是很喜欢代码，以前是零基础，之前的计概课程没选上python，学的c语言，但是基本没有接触什么算法，不是很清楚自己学计概是怎么过来的，比较难绷。

到了数算，掌握基本的代码语法消耗了一定时间，基本全程不是很能跟上课程学习的进度。自己练习也不足。期末机考前认真准备了一阵子，勉强了解了dp，dfs，bfs，还有拓扑排序等图的算法，记下了各种数据结构，如链表，栈等。在机考的时候前半小时做了两道题，结果第三道题卡到死后面的题一眼没看，原来是题目理解错了，问助教的时候考试都要结束了，助教给全体同学解释的时候也有其他同学惊呼，感觉不完全是自己的问题，机考释怀了。自然也是心态炸了，最后十分钟没想起来自己最开始的思路会导致哪里出错，考完试回寝室20分钟不到重新写了一次ac掉了。

这么一来，现在回忆这学期学了啥愣是啥都记不起来，光记得助教那句“这道题不想卡大家的”，我也蛮想拿高分的。

代码感觉就是跟着ai学吧，有各种标准的写法，但有的时候过于繁杂；力扣的解答通常都比较简洁，B站上的一些教程也很有帮助。

闫老师的课程也很好，跟我的计概课程差别蛮大的，我当时快期末了上课还在讲几种排序，二分法之类的，这学期也算是学了不少东西，非常感谢自己选了这门课。
